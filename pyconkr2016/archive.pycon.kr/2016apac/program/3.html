Description

자신의 스마트폰으로 찍은 사진 데이터로부터 얻을수 있는 시간, 위치, 얼굴인식(이미지 분석) 데이터를 추출하여 위치, 이동, 여행, 감성 등의 정보를 재해석하는 데이터 분석 과정을 통해 자신도 몰랐던(잊었던) 것을 데이터로 표현하는 과정과 결과에 대하여 공유할 예정입니다. 
* 여행 중 찍은 사진으로만 분석된 "베트남 여행의 재구성"이라는 저의 블로그 글을 참고하시면 이번에 발표 지원하는 내용을 가늠해 볼수 있을듯하며, 세부적인 내용은 아래와 같이 될 예정입니다. 

데이터 랭글링
  - EXIF데이터 추출 
  - Vision API 사용하기 
  - 시간/거리 기반 데이터 클러스터링/세그멘테이션
Life 트랙킹과 데이터 해석
   - 데이터의 통계적 탐색 
   - 위치 트랙킹 (타임라인)
   - 네트워크 분석 
   - 이미지 감정 분석 
Jupyter Notebook으로 데이터를 표현하는 방법에 대하여
    - 데이터와 적절히 차트 선택( histogram, scatter, violine plote 등)
    - 네트워크 그래프 
    - Javascript의 활용 

이렇게 파이썬을 이루어지는 데이터 분석의 전체 과정과 사진 데이터를 자기 자신을 정량적으로 해석하는 
자아 정량화(Quanfied Self) 결과를 공유할 예정입니다. 

감사합니다.
