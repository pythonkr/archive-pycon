Description

컴퓨터 프로그래밍을 배우는 것은

남녀노소 모두에게 재미있고, 즐거워야 합니다.

ㅡ RUR-PLE(러플)

리보그란 격자 세상속 로봇에게 주어진 일을 파이썬 프로그래밍으로 해결하면서 문제 해결력을 기르는것을 목적으로 하는 언어입니다.

이 발표에서는 리보그에 대해 전반적으로 소개드리고 리보그와 파이썬 프로그래밍으로 문제 해결력을 익히는 과정을 간단한 예제 한 문제를 풀면서 설명합니다.

마지막으로 파이썬의 특징 중 하나인 객체지향을 리보그에 적용시키는 방법을 소개해드립니다.


혹시 파이썬 프로그래밍 교육으로 많이 쓰이는 러플을 아시다면 러플의 2세대 온라인 버전이라고 이해하시면 됩니다 :D

이번 파이콘으로 파이썬을 쉽게 배우기 위한 교육용 도구의 정보 교류가 이루어지는 계기가 됐으면 합니다.

청중에게 도움 되는 자료

  • 리보그 홈페이지 : http://reeborg.ca
  • 리보그 깃허브 저장소 : https://github.com/aroberge/reeborg

발표가 진행되는 순서

  • 리보그 소개
    • 리보그의 역사 : 1981 ~
  • 리보그 사용자 환경 소개
    • 라이브러리 사용하기
    • 기본 명령어 소개
  • 문제 해결 : 리보그에게 일 맡겨보기
  • 객체 지향 다루기
    • 객체 지향 코드 예제
    • 내부 속성, 메서드 보기
    • 자바스크립트
