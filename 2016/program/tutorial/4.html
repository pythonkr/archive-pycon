Tutorial Beginner 4 hours Korean 45 people

Brief

Write the Docs은 소프트웨어 문서화를 다루는 커뮤니티입니다. Write the Docs의 서울 Meetup을 파이콘에서 진행하려고 합니다.

Detailed description (required)

****** 참여해주신 모든 분들 덕분에 Write the Docs Seoul Meetup #1는 성황리에 종료되었습니다. ******


커뮤니티 메인 사이트: http://www.writethedocs.org/

Meetup: http://www.meetup.com/write-the-docs-seoul/

페이스북: https://www.facebook.com/groups/1694859810774599/

트위터: https://twitter.com/WTDSeoul

슬랙(#seoul): http://slack.writethedocs.org/


Write the Docs은 소프트웨어 문서화를 다루는 커뮤니티입니다. Write the Docs의 첫 서울 Meetup을 파이콘 APAC 2016에서 진행합니다. 이 모임에서는 소프트웨어 문서화, 글쓰기, 번역을 주제로 프로그래머, 편집자, 번역가 등이 모여 교류하는 시간을 가지려고 합니다. 이와 관련된 주제와 관심 있는 분들이라면 누구나 참석 가능합니다. 문서화에 대해 대화를 나누고 서로의 생각을 교류하는 시간을 함께 만들어주세요.


일시: 2016년 8월 15일 10시 - 2시

장소: 아카마이 코리아 (강남역 메리츠타워 21층)


파트 1: 주제 발표

  • 10시 10분 ~ 10시 20분
  • Write the Docs 소개(10분)


  • 10시 20분 ~ 11시
  • 프로그래밍 책을 쓰기 위한 환경 구성과 책을 잘 쓰기 위한 도구(20분) - 이지호
  • 인디자인을 활용한 자동 조판 워크플로우 실험(20분) - 김승호


  • 11시 10분 ~ 12시 10분
  • 문서 국제화(20분) - 석우징
  • 디지털 미디어 네이티브 문서: Explorable Explanations(20분) - 김대권
  • 글을 잘 쓰고 싶은 사람들을 위한 워드 사용법(20분) - monaca


파트 2: 대화테이블

12시 30분 ~ 2시

세션 발표가 끝난 후에는 참가자 분들이 주제를 정해 이야기할 수 있는 시간을 가질 예정입니다. 발표 주제를 비롯해 참가자 분들은 자유롭게 이야기하고 싶은 주제를 제출할 수 있습니다. 사람들은 자신이 토론하고 싶은 주제를 정해서 모이고 해당 주제로 자유롭게 이야기를 나눕니다. 정해진 시간이 지나면 각 테이블 별로 이야기한 내용을 요약해서 발표하고 공유하는 시간을 가집니다.
