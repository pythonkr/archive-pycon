파이썬으로 기초 산수 풀어보기

Tutorial Beginner 2 hours Korean 45 people

윤석준 바벨피쉬

Brief

파이썬으로 기초 산수 문제를 풀어보고 그래프로 표현하는 방법에 대한 따라하기 식의 튜토리얼

Detailed description (required)

페이스북 바벨피쉬에서 진행중인 스터디인 복잡복작밑바닥수학에서 공부하고 있는 내용중 파이썬 으로 수학문제 풀이에 필요한 가장 기초적인 부분에 대해서 실습으로 진행합니다.

sympy, matplotlib을 이용하여 파이썬으로 기초 산수 문제를 풀어보고 그래프로 표현하는 방법에 대한 따라하기 식의 튜토리얼로 진행됩니다.

파이썬으로 풀어보는 수학 (http://www.acornpub.co.kr/book/doing-math-with-python) 책의 내용으로 진행합니다.


* 필요사항

  - Jupyter notebook (Python 3.xx)을 이용해서 진행할 예정이니 설치해 오시기 바랍니다.

  - matplotlib 도 같이 설치해 오세요. ( pip install matplotlib-venn )

  - 원도우즈 사용자 분들 중 사용자명이 한글인 경우 실습이 원활하지 않을 수 있습니다. ( C:/Users/한글이름)


* 발표자

  - 윤석준 :  2장(matplotlib)과 4장(sympy)

  - 심상진 : 7장(미적분) 


* 실습파일 (.ipynb)

- https://github.com/DevStarSJ/Study/tree/master/Blog/Python/DoingMathWithPython

- 직접 입력하면서 따라하시기를 추천해드립니다.
