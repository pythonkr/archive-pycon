Flask로 시작하는 웹 프로그래밍

Intermediate 4 hours Korean 45 people

이지호 한국방송통신대학교 대학원 정보과학과

Date and Venue

일시 : 2017년 8월 14일

시간 : 오후 1시 ~ 오후 5시

장소 : 코엑스 그랜드볼륨 2층 

신청 : https://onoffmix.com/event/107237

Brief

Flask 튜토리얼을 시작할때쯤 되어...
지난 2017년 7월 18일 오전 0시 30분쯤 되어 파이콘 준비위원회 조성수님이 제게 페메를 주셨습니다.

"무슨 일로"
제게 플라스크 튜토리얼을 진행해달라 하십니다. 아주 화끈히 응답했습니다. "그러시죠"

짧은 제 소개
- 우주 정복을 노리고 있으나 현실은 집 근처 10km을 못 벗어나는 인생 초보러. 2주에 한번 돌아오는 헌혈이 세상을 구성하는 물질의 20%라고 생각한다. 대학을 졸업하며 "다시 공부안해!!!" 해놓고 대학원 가서 "세상을 혁신하는 논문 써야지!!!"라면서 "논문보다 책 집필!!!"을 시전하는 이 시대의 직장에 다니는 잉여!

조성수 만세!! 이렇게 시작하게 되었습니다.

여러분 이 파이썬을 좋아하든 그렇지 않든 웹 프로그래밍은 파이썬을 사용해, 그리고 플라스크를 사용해서 시작해서 플라스크로 끝내시길 바랍니다.

이 세션에선 제가 집필한 책의 내용을 기반으로 하며 튜토리얼에 참여하는 참가자들이 실무에서 파이썬으로 스스럼없이 웹 프로그래밍을 플라스크로 할 수 있기를 기대합니다.

Detailed description (required)

플라스크와 함께할 내용...

1. 왜 파이썬과 플라스크인가?
2. 웹 프로그램 플라스크로 5분만에 시작하기
3. 플라스크로 시작하는 20분 게시판
4. 플라스크와 Database 님들과 함께하기
5. 원하는대로 말하는대로 HTTP Session을 자유자재로
6. Vue.JS와 함께하는 플라스크. 이렇게 자유로와도 되는거야?
7. 슬랙과 텔레그램과 함께하는 알림 이야기

2017 파이콘! 저와 함께하는 플라스크를 사용하는 이야기에 깊게 고요하게 빠져봅시다!
