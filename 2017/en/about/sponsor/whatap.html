WhaTap

  • Gold
  • https://www.whatap.io/

WhaTap은 "어떻게 하면 개발자가 더 멋진 서비스를 만들 수 있을까?"라는 물음에서 시작했습니다.

빠르게 변화하는 IT 환경에서 개발과 운영의 경계가 희미해지며 DevOps라는 환경이 등장했습니다. 개발자는 개발뿐만 아니라 서비스를 운영하기도 하면서 더 나은 서비스를 위해 더 다양한 노력을 쏟게 되었습니다. 개발자들로 구성된 WhaTap은 개발자들에게 보탬이 되고자 Python Application Monitoring이라는 서비스를 만들었습니다.

Python Application Monitoring을 통해 사용자가 서비스에 접속했을 때 어떤 일들이 벌어지고 있는지 눈으로 확인할 수 있습니다. 트랜잭션의 응답시간, 수행한 쿼리, 외부 API Call 등을 모두 프로파일링합니다. Python Application Monitoring으로 확보한 데이터를 기반으로 서비스를 향상시키는 중대한 의사결정을 수행할 수 있습니다.

그래서 개발자들이 더 멋진 서비스를 만드는 데 보탬이 되었으면 합니다.

파이썬 커뮤니티에서 열정적으로 교류하며 정보를 공유하고, 성장하는 여러분 모두를 응원합니다.
