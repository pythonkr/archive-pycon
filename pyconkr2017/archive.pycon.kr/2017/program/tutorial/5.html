나만의 웹 크롤러 만들기

초급 4 시간 한국어 10 명

이준범 GeekHub

- DjangoGirls Seoul 

- <나만의 웹 크롤러 만들기> 시리즈 연재

- Py + Django = <3

일시와 장소

일시 : 2017년 8월 15일

시간 : 오후 2시 ~ 오후 6시

장소 : 코엑스 그랜드볼륨 2층 

신청 : https://onoffmix.com/event/107244

간단한 설명

Python3/requests/BeautifulSoup/Selenium을 이용한 웹 크롤러 만들기 실습 튜토리얼입니다.

자세한 설명

* 다루는 내용 *

- requests + BeautifulSoup를 이용한 공개된 웹 사이트 크롤링

- requests + BeautifulSoup를 이용한 간단한 로그인

- selenium + chromedriver + BeautifulSoup를 이용한 로그인과 크롤링


* 예상 참가자 *

파이썬을 다뤄보았고 list/dict를 다뤄본 초급자.


* 예상 진행 시간 *

(15분) 웹 크롤링이란? + HTML의 구조와 CSS Selector, 그리고 HTTP Method의 이해 

(15분) requests 모듈과 BeautifulSoup4 소개 및 간단한 조작법 안내

(50분) 실습: 클리앙 크롤링, 로그인

(N분) Intermission & QnA

(15분) requests + BeautifulSoup으로 크롤링하지 못하는 경우 소개

(20분) Selenium와 ChromeDriver소개 및 간단한 조작법 안내

(50분) 실습: 네이버 로그인 후 카페 내 검색 결과 크롤링하기

(N분) QnA

* 추가사항 *

실습 대상 사이트는 참석자의 수준과 요청에 따라 변경될 수 있습니다.

실습이 빨리 끝난 경우 추가 사이트를 함께 분석하는 시간을 가져볼 예정입니다.
