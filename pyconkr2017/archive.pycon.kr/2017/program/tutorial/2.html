python을 이용하여 google api와 공공데이터 포털 api 사용하기

초급 4 시간 한국어 10 명

Bill Park GeekHub

부산에서 서식중인 메이커입니다.

일시와 장소

일시 : 2017년 8월 15일

시간 : 오전 10시 ~ 오후 2시

장소 : 코엑스 그랜드볼륨 2층 

신청 : https://onoffmix.com/event/107247

간단한 설명

python 3 를 이용하여 google api 와 공공데이터 포털의 api를 사용하는 과정입니다.
api 결과값을 이용하여 간단한 telegram bot까지 만들어 볼 예정입니다.

자세한 설명

구글의 공식 Documents가 있지만 이것만으로는 하기가 힘들어 많은 노력(삽질)이 필요한 구글 api

날씨 자료(기상청 제공)와 미세먼지 자료(에어코리아 제공)를 가져올 수 있는 공공데이터 포털의 api를 사용해 볼 계획입니다.


진행순서

1. google calendar api 사용하기 - 일정 가져오기, 일정 만들기

2. google drive api 사용하기 - 파일 다운받기, 이미지 업로드하기

3. google url shorten 사용하기 - drive에 업로드 한 이미지 링크 만들기
4. 공공데이터 포털 날씨 데이터 가져오기 - 온도, 습도, 강수량, 하늘상태

5. 미세먼지 데이터 가져오기 - PM2.5, PM 10 농도

6. 데이터를 이용하여 Telegram bot 만들기


미리 공공데이터 포털(www.data.go.kr) 회원가입을 해오시면 시간을 절약할 수 있습니다.

휴대폰(혹은 컴퓨터)에 Telegram 어플을 깔아오셔야 bot을 사용할 수 있습니다.
