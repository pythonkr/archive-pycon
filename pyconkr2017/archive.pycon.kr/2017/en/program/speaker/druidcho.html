식품과 온라인 서비스에 관심이 많고 (주로) 웹페이지를 포함한 Server-side를 하는 개발자입니다.


산업기능요원으로 GIS 관련 업체에서 Javascript 기반 웹 지도 엔진 유지보수 업무로 개발에 입문하였고, 그후 AwesomePiece라는 모바일 게임 개발사에서 약 2년간 Python-Django 기반 비동기 게임서버, Java-Netty 기반의 실시간 동기 게입서버개발자로 일하였습니다.


2013년 INTAKE 라는 식품회사를 창업하여, 현재까지도 그 회사에서 일하고 있습니다.


아래 내용들을 잘 알고, 주로 사용 합니다.


Languages : Python, Java, Javascript

Framework/Library : Django, Flask, Celery, Netty

Cloud Platform : AWS(EC2, ELB, ElasticBeanstalk, RDS, ElastiCache, S3/Cloudfront, Lambda) / AWS Solutions Architect-Associate(SAA)

Tools : Git,  Google Analytics / GAIQ
