Django admin site를 커스텀하여 적극적으로 활용하기

Slides

Description

많은 스타트업에서 django를 활용하여 Back-end API나 Website를 구축합니다. 운영하는 서비스(모바일앱 혹은 웹사이트)를 통해 축적한 데이터를 시각화하는 Dashboard나 콘텐츠를 관리할 수 있는 툴(CMS)을 개발하여 꾸준히 분석해야 합니다. Django를 활용하여 직접 관리자용 웹사이트를 개발할 수 있지만, 기존에 개발되어 있는 admin site를 커스텀하여 사용하면 더 좋은 결과물을 만들어낼 수 있습니다. 대표적으로 Custom UI/Layout, Custom Filter(데이터 정렬 관련), Custom Action(데이터 필터링 후 추가 작업), Custom page(Model page 외에 추가 가능한 일반 페이지)가 있습니다. 새로운 기술이나 라이브러리를 소개하는 것이 아니라, 장고 공식 문서에서 소개하는 방법을 통해서 admin site를 필요에 맞게 커스텀해보는 여러가지 방법을 소개하려고 합니다. 단순히 소개에 그치지 않고, custom 기능들을 활용했을 때 어떤 결과물을 낼 수 있는지도 스크린샷을 통해 소개할 예정입니다.
