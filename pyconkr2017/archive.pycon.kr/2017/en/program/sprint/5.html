Sprint Leader

Apache Zeppelin 커미터로 활동하며 더 나은 사용성과 건강한 커뮤니티를 위해 노력하고 있습니다.

Brief about project

Apache Zeppelin 은 빅데이터 세계에서 Visualization 을 담당하는 오픈소스 Notebook 프로젝트입니다. 아마 파이썬을 주로 사용하시는 분들은 Jupyter 가 익숙할테지만 커다란 데이터를 처리해야 하거나 Apache Spark 를 쓰신다면 Zeppelin 을 이미 써보셨거나 들어보셨을 수도 있을거에요.

본 스프린트에서는 데이터셋을 이용해 제플린 위에서

- Tensorflow 로 모델을 만드는 과정을 진행해 보고
- 그 결과를 Python 을 이용해 Visualization 해 볼 예정입니다.

이 과정에서 참가자들은
- Bug 를 찾아 고치거나
- 잘못되거나 부족한 Documentation 을 고칠 수 있고
- 필요하다고 생각되는 Feature 를 제안하는

Pull Request 를 만들고 커뮤니티의 Review 를 받는 과정을 통해 오픈소스에 기여하는 값진 경험을 얻어갈 수 있습니다.

*참고* 아래 링크를 따라 미리 스프린트 전에 빌드 환경을 갖추어 주시면 정말 빠른 진행에 도움이 됩니다. (빌드에 시간을 쏟느라, PR 를 만들지 못하거나 Tensorflow 실습을 할 수 없다면 우리는 슬플거에요..)
