Brief about project

asyncio 기반의 Docker API client 라이브러리입니다. (Python 3.5 이상 지원)
이번 스프린트에서는 그동안 밀려있는 문서화, 리팩토링, 테스트 개선 작업을 진행할 예정입니다.
개발과정에서의 토의는 영어와 한국어로 가능하나, GitHub 이슈 및 문서화는 모두 영어만 사용합니다.
영어 토의나 문서 작성이 어려운 분들은 진행자가 도와드립니다.

It is an asyncio-based Docker API client library targeting Python 3.5 or higher.
This sprint will cover pending improvements of documentation, refactoring, and testing.
We can have discussions in both English and Korean but you need to use English only for writing GitHub issues, commit messages, and documentations.
I will help you talk and write in English if needed.

업데이트) aiodocker 말고도 aiotools 프로젝트에 대한 스프린트도 함께 준비할 예정입니다!
Update) I will cover aiotools project as well as aiodocker!
