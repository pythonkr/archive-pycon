Python 테스트 시작하기

Tweet

  • Korean
  • 국제회의실
  • 이호성 / hosung
  • Photography and recording is allowed

Presentation slides

http://www.slideshare.net/hosunglee948/python-52222334

Description

개발자들은 항상 테스트를 작성합니다. (그렇지 않다면 해야 합니다.) 하지만 좋은 테스트를 작성하는 것 뿐 아니라 프로젝트의 완료까지 테스트를 지속하는 것도 쉽지 않은 일입니다. 다음 내용들을 소개해 테스트에 익숙하지 않은 초보 개발자분들께 가이드라인을 제시하고자 합니다.

  • 테스트가 왜 필요할까요? 어떤 테스트가 필요할까요?
  • 좋은 테스트란 무엇인가요?
  • unittest 소개 및 활용
  • 테스트 관련 툴 소개
  • 제가 하고 있는 테스팅 과정 소개

위 내용들을 신입 후배와 페어 프로그래밍을 하는 기분으로 소개하려 합니다.
