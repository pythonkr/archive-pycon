R이 판치는 세상에 Python 데이터 분석가로 사는 법

Tweet
  • Data / Analysis / Machine Learning
  • Korean
  • 국제회의실
  • 하용호 / yongho
  • Photography and recording is not allowed

Description

데이터 분석의 시대라고 합니다. 그런데 서점에 가서 데이터 분석 관련 책들을 찾아보면 전부 R로 되어 있어요. 나의 파이썬이 이럴리가 없어. 외국에서는 데이터분석할 때 파이썬 많이 쓴다고 하는데, 어째서 우리나라에서는 이렇게 R만 보이는거냐! R쓰는 친구가, 이제 그만 반항하고 R을 배우라고 말해온다해도 우리는 질 수 없습니다. Python은 훌륭해요. 멋져요. 아름다워요. 우리는 R이 안되는 걸 합시다. Python으로 데이터 분석하면서 아. R이 아니라 Python이라서 다행이야. 라고 느낄만한 순간들을 만나봅시다. 요즘 뜨는 Spark과 ipython notebook을 붙여서, interactive하게 클러스터를 다뤄 봅시다. spark과 pandas를 오가며 R은 로딩도 꿈꾸지 못할 애들을 주물러 봅시다.  솔직히 R의 차트가 더 이쁜거 같다구요? 더이상 구린 컬러의 matplotlib은 이제 그만. 여러 그래픽 라이브러리들로 R의 ggplot보다 더 멋지게 그래프를 그려봅시다. 이제 친구에게 말할 시간입니다. ‘어? R은 이거 안되냐?’
