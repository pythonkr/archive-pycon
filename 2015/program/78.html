Lightning Talk

Tweet
  • 2015-08-30 (Sun) 18:00 - 19:00
  • Korean
  • 국제회의실
  • Photography and recording is allowed

Description

  1. 차영호 - clang 과 javaparser와 python을 이용한 소스변경사항 고객용 리포트 자동생성하기
  2. 서지혁 - 네이버 블로그를 백업해주는 도구인 exitnaver와, exitnaver
  3. 박조은 - 어느 여자개발자의 육아휴직 2년
  4. 임재수 - built-in 모듈인 ctypes에 대해 소개할까 합니다.
  5. 김영근 - PyCon의 철학과 다양한 파이콘 참석 후기.
  6. 이비오 - 가제 : 우리는 이래도 되는가?
  7. 서정민 - 개밥먹기 : 커뮤니티의 개발과 운영
  8. 김대현 - C++에서 닷넷으로 그리고 자바, 파이썬. 파이썬이 내게 가져다 준 것들.
  9. 이지호 - 기술 서적을 집필하기 위해 필요한 동기와 자잘한 기술
  10. 강대성 - 엔지니어의 삶 - Raspberry Pi 와 Python 으로 에어컨 제어
  11. 박현우 - 4.5cm ~ 8.5m
