Brief

Kaggle 대회에 필요한 실전 know-how

Detailed description (required)

﻿﻿점수가 올라갈 듯, 말 듯 마음처럼 안 풀리는 Kaggle Competition!


지난 2년간 맨땅에 헤딩하면서 배운 Kaggle

많은 대회를 참가했다고 할 수 없지만 Titanic부터 TwoSigma:Renthop, Benz, Zillow Prize, Quora Question Pair 등의 대회를 참가하면서

배운 경험과 대회 끝나고 분석한 우승자 솔루션을 기반으로 Kaggle 대회에 필요한 know-how를 알려드립니다.

(마침 Coursera에 How to Win a Data Science Competition: Learn from Top Kagglers 강의도 있어 이 내용도 같이 진행하도록 하겠습니다.)


튜토리얼 대상

  • ﻿Titanic같은 Tutorial 대회를 마치고 실제 Competition에 참여하고 싶으신 분!
  • 지금은 잘 모르더라도 나중에 꾸준히 Kaggle이나 Data Science Competition에 참여하실 분!


튜토리얼 내용

※ 각 항목마다 별도 실습이 존재합니다.

  • ﻿Feature Engineering 기초 (Feature Type별 특성을 이해하고 각 Type별로 Feature를 추가하는 방법 학습, Tree 모델과 Non-Tree 모델에서의 Feature Engineering 차이점 학습)     
  • Feature Engineering 고급 (Cross-Validation 전략, Data Leakeage 사례 공유, 다양한 EDA 사례 공유, Mean Encoding)
  • ﻿Tree 기반 알고리즘 소개와 Parameter Tuning
  • 우승자 솔루션 따라하기 (GrandMaster들의 대회 Pipeline을 소개하고 OOF Feature를 이용한 Stacking 실습)


준비사항

  • 준비물: 개인용 노트북 ( Kaggle 가입 필수!)
