아이돌봄

파이콘 본 세션이 있는 8월 18일 토요일과 19일 일요일에 육아로 인해 파이콘에 참여가 어려운 분들을 위해 아이돌봄을 운영합니다. 알레르기가 없는 아이들에 한 해 오전과 오후에 한 번씩 주스와 함께 간단한 간식이 제공 되며 점심식사는 제공되지 않습니다. 아이들의 점심식사는 부모님께서 직접 도시락이나 인근식당을 이용해 주셔야 하며, 206A호에 별도의 휴게공간이 제공됩니다. 206A호에서 배우자와 함께 간식이나 식사, 수유, 낮잠, 기저귀 교체 용도로 자유롭게 이용이 가능합니다.


양일 간 돌봄을 원하시면 양일 모두 등록해 주셔야 합니다.

1일차 등록

  • 36개월 미만 영유아 : 신청하기
  • 36개월 이상 ~ 미취학 아동 : 신청하기

2일차 등록

  • 36개월 미만 영유아 : 신청하기
  • 36개월 이상 ~ 미취학 아동 : 신청하기

  • 대상 : 파이콘 티켓 구매자(튜토리얼과 스프린트 제외), 파이콘 접수 이메일 확인

  • 연령 : 미취학 유아동

  • 일정 : 2018년 8월 18일(토) 10:00~ 17:00,  19일(일) 10:00~ 17:00

  • 식사 : 간식은 제공되나 점심식사는 제공하지 않으며, 부모님께서 도시락이나 인근 식당을 이용해 주셔야 합니다.

  • 준비물 : 낮잠용 이불, 간식(알레르기 등의 이유로 별도 간식이 필요한 경우와 24개월 미만의 유아), 양치 도구, 기저귀, 여벌 옷, 이유식(필요한 유아에만 해당), 코엑스 실내온도가 낮아 긴팔과 긴바지도 함께 준비해 주셔야 합니다.

  • 등록 접수 시 필요 정보 : 연령, 성별, 형제자매 여부, 알레르기 여부, 24개월 미만의 경우 개월 수

  • 참가비 : 한 명의 아동 당 1일 1.5만원 - 쌍둥이나 형제자매는 인원 수 만큼의 비용 발생, 2일 모두 이용할 경우 한 아동 당 3만원의 비용이 발생

  • 주차권 : 종일권 50% 할인 구매 가능

  • 모집 인원 :

    • 36개월 미만 영유아 - 10명 (돌봄 선생님 3명)

    • 36개월 이상~ 미취학 아동 - 20명 (돌봄 선생님 4명)

  • 환불조치 안내 : 행사당일 37.5도 이상의 고열이 발생하거나 전염성 질병이 의심되는 경우 전액 환불조치하고 돌봄 이용이 불가합니다.
